# Octalang 文法書

## 1. 基本思想

この言語は、**「7種の記号（3bit）による超高効率通信」** と **「パーサー（解析機）の負荷最小化」** を目的として設計された人工言語である。

- **文字セット:** `a`, `i`, `u`, `p`, `t`, `k` とスペース。
- **データ表現:** 1文字あたり3bitで表現可能。

## 2. 単語と品詞

品詞は単語の構造や文脈によって決まる。

### 2.1. 基本単語

基本となる単語は、定義済みのリストによって与えられる。

- **代名詞 (1文字):**
    - `a`: 私
    - `i`: あなた
    - `u`: それ

- **動詞 (2文字):** 動作や状態を表す。
    - `ap`: 持つ / ある
    - `ip`: 与える
    - `pp`: する / 作る
    - `pt`: 言う
    - `pk`: 食べる

- **名詞 (3文字):** 事物や概念を表す。
    - `pap`: 食べ物
    - `pat`: 人
    - `pup`: 水
    - `kap`: 家
    - `kak`: お金

- **前置詞 (1文字):** 格を示すマーカーとして機能する。
    - `p`: 場所 (`at`, `in`)
    - `t`: 方向・終点 (`to`)
    - `k`: 起点 (`from`)

### 2.2. 接尾辞による品詞派生

基本単語に接尾辞を付けることで、品詞を変化させたり、意味を追加したりできる。

| 接尾辞 | 機能 | 接続先 | 例 |
| :--- | :--- | :--- | :--- |
| **-a** | **形容詞化 / 所有格化** | 名詞 | `pup` (水) -> `pupa` (水の、液体の) |
| **-i** | **主語化** | 名詞 | `pat` (人) -> `pati` (人が) |
| **-k** | **使役・受動** | 動詞 | `pk` (食べる) -> `pkk` (食べさせる) |
| **-p** | **否定** | 動詞 | `pt` (言う) -> `ptp` (言わない) |
| **-t** | **過去形** | 動詞 | `ap` (持つ) -> `apt` (持っていた) |
| **-u** | **名詞化** | 動詞 | `kt` (痛む) -> `ktu` (痛み) |

## 3. 文法構造 (Syntax)

### 3.1. 基本語順

基本は **`主語 + 動詞 + 目的語`** のSVO型となる。かならず他動詞を用いる。

**基本フォーマット:**
> **`S` `V` `O`**

- **S (主語):** 接尾辞`-i`が付いた名詞。
- **V (動詞):** 動詞。
- **O (目的語):** 名詞や代名詞。

### 3.2. 前置詞句

前置詞 (`p`, `t`, `k`) は、名詞の前に置かれ、場所や方向などの追加情報を示す。

> **`前置詞` `名詞`**

- `p kap`: 家で
- `t tut`: 未来へ
- `k tip`: 過去から

### 3.3. 疑問文

- 疑問文: 動詞を文頭に置くことで表現する。
    - `ap ii kak?` (あなたは お金を 持っていますか？)
