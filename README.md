# Octalang

Octalangは、**「7種の記号によるデータの軽量化」** を目的として設計された人工言語です。

## 特徴

- **最小限の文字セット:** `a`, `i`, `u`, `p`, `t`, `k` とスペースのみで構成されます。
- **高圧縮率:** 各文字を3bitで表現できるため、非常にコンパクトなデータ表現が可能です。
- **シンプルな文法:** `主語 + 動詞 + 目的語` (SVO) を基本とし、前置詞をつなげる単純な仕組みです。

## 基本的な文の例

- **私はそれをする。**
  - `ai pp u`
    - `ai`: 私 (主語)
    - `pp`: する (動詞)
    - `u`: それ (目的語)

- **私は家でそれをする。**
  - `ai pp u p kap`
    - `p kap`: 家で (前置詞句)
